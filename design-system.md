# My Design System

## Color System

I used a primary blue color for buttons and important elements.

I added a secondary orange color for hover states and small highlights so the design is not too plain.

The background is light to keep everything easy to read.

## Typography

I used Arial because it is simple and beginner-friendly.

I used different sizes for headings and paragraphs to create a clear hierarchy.

## Spacing

I used a simple spacing system (8px, 16px, 24px) to keep everything consistent.

## Components

### Button

The button uses the primary color and changes on hover.

It has padding and rounded corners to look cleaner.

### Card

The card has a white background, padding, and rounded corners.

I used cards to organize the content.

## AI Improvements

I added a hover state to the button and kept my variable names simple and consistent.

## Design Principles

### Consistency

I reused the same colors, spacing, and fonts across the page.

### Accessibility

I made sure the text is readable and added hover states.

### Clarity

I kept the layout simple and organized.
